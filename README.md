\# Asymétrie d'Information \& Efficience des Marchés Financiers



Étude empirique reproduisant la méthodologie d'El Bouazizi (2018)  

sur des données du marché américain (S\&P 500, 2015–2024).



\## Structure

\- `rapport.Rmd` : analyse complète en R Markdown

\- `data/` : données financières

\- `presentation/` : diaporama final

\- `output/` : graphiques et tableaux



\## Packages R requis

quantmod, tidyverse, ggplot2, broom, kableExtra
